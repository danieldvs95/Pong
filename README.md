# Project Pong

Retro arcade 2D game develop based on Atari's Pong.

Developed with html5 and vanilla javascript.

<img alt="Neon Skylines footage" src="https://media.giphy.com/media/l4EoNyoN5lPj551Je/giphy.gif" width="100%">

# Controls :

- P1

  Up : W\n
  Down : S

- P2

  Up : Up arrow\n
  Down : Down arrow
